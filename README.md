A responsive and modern calculator built using HTML, CSS, and JavaScript.

- Responsive design
- Clean and modern UI

**Technologies Used:**
- HTML5
- CSS3
- JavaScript

### ✅ Task 3 – To-Do Web App
A task management application that helps users organize daily activities.

**Features:**
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Filter tasks
- Local Storage support
- Dashboard counters
- Responsive design

**Technologies Used:**
- HTML5
- CSS3
- JavaScript
- Local Storage API

---

### 🔐 Task 4 – Login Authentication System
A secure frontend authentication system using Local Storage.

**Features:**
- User Registration
- User Login
- Password validation
- SHA-256 password hashing
- Duplicate email/username validation
- Protected Dashboard
- Session Management
- Logout functionality
- Responsive UI

**Technologies Used:**
- HTML5
- CSS3
- JavaScript
- Local Storage API
- Web Crypto API (SHA-256)
